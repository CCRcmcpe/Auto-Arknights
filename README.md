## 入门

> [最新版本下载](../../releases/latest)  
> [使用手册](../../wiki)

### 必须运行要求

* CPU 支持 AVX2 指令集。
* Windows 7 x64 或更高版本的操作系统。
* .NET 5.0 运行时，[官方下载](https://dotnet.microsoft.com/download/dotnet/current/runtime)。

### 注意事项

* 本程序不支持国服以外的明日方舟区服。
* 原理上，使用本程序没有被判定为作弊的风险；但是作者不对使用此程序造成的任何损失负责。
